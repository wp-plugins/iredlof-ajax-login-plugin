== Description ==

iRedlof Ajax Login adds a beautiful Javascript loaded Login/Register top panel to your wordpress theme.This plugin is based on my iRedlof Inspire Theme. Checkout the toppanel demo on <a href='http://rlchandani.co.cc'>http://rlchandani.co.cc</a> .

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done

== Frequently Asked Questions ==

= My theme is distorted after activating this plugin ? =

The layout this plugin injects in your current theme might me making it distorted, Change your theme to use this plugin else de-activate it from plugin menu in admin panel.

= Plugin Shows up but its not working ? =

Your theme might be using jQuery javascript library. So there might a conflict between your theme javascript functions and this plugins javascript functions.Change your theme to use this plugin else de-activate it from plugin menu in admin panel.
